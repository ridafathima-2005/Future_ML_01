Sales & Demand Forecasting for Businesses (Task 1)

Project Overview

This project focuses on forecasting future sales using historical business data.
It applies machine learning techniques to analyze trends and generate predictions.

---

Objective

* Perform time-series analysis
* Build a forecasting model
* Generate business insights from sales data

---

Features

✔ Data cleaning and preprocessing
✔ Time-based feature engineering (year, month, day)
✔ Sales forecasting using Linear Regression
✔ Model evaluation using MAE and R² score
✔ Visualization of actual vs predicted sales
✔ Future sales prediction

---

Tech Stack

* Python
* Pandas
* Matplotlib
* Scikit-learn
* Jupyter Notebook

---

Output

* Sales trend over time
* Predicted vs actual sales comparison
* Forecasted future sales values

---

 How to Run

1. Install dependencies:

```bash
pip install pandas matplotlib scikit-learn
```

2. Open the notebook:

```bash
Sales & Demand Forecasting for Businesses.ipynb
```

3. Run all cells
